***Quantity Measurement App – Frontend (Angular)***

A modern Angular-based UI for the Quantity Measurement Application that allows users to perform unit conversions, arithmetic operations, and manage history with authentication support.

***Tech Stack***
Angular 21
TypeScript
HTML5 & CSS3
RxJS
Angular Standalone Components
REST API Integration (Spring Boot Backend)
